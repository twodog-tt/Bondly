# Bondly 智能合约文档

## 项目概述

Bondly 是一个 Web3 去中心化社交价值网络平台，包含多个智能合约模块。

## 合约列表

- [BondlyRegistry](./BondlyRegistry.md) - BondlyRegistry 合约
- [BondlyToken](./BondlyToken.md) - BondlyToken

## 快速开始

1. 查看具体合约文档了解功能
2. 使用 `node scripts/generate-docs.js` 重新生成文档
3. 所有文档基于 NatSpec 注释自动生成

## 合约架构

- **Token**: 代币相关合约
- **Governance**: 治理相关合约  
- **Staking**: 质押相关合约
- **NFT**: NFT 相关合约
- **Registry**: 注册表合约
